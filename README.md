# Earth Insights - NASA Terra Mission Platform

## Overview

**Earth Insights** is an interactive, educational web platform dedicated to the NASA Terra satellite mission and Earth observation science. Launched in 1999, Terra represents 25 years of pioneering satellite science dedicated to understanding our planet's climate, atmosphere, oceans, and ecosystems.

Earth Insights transforms complex NASA Earth observation data into engaging, interactive experiences. Through immersive simulations, educational content, and decision-making games, the platform makes satellite science accessible to students, educators, researchers, and the general public.

### Mission Statement

To democratize Earth observation science by providing an engaging, interactive platform that helps users understand Terra's groundbreaking contributions to climate science, environmental monitoring, and planetary research.

## Key Features

### 1. Interactive Video Gallery
- **Comprehensive Video Library**: Curated collection of NASA Terra mission videos
- **Multi-Format Support**: Educational documentaries, technical briefings, and visual explanations
- **Responsive Player**: Embedded video player optimized for all devices
- **Video Categorization**: Organized by topic (Climate, Atmosphere, Oceans, Ecosystems)
- **Metadata Display**: Title, duration, topic tags, and descriptions
- **Smooth Navigation**: Seamless video browsing experience with fluid animations
- **Educational Context**: Each video paired with scientific background information

### 2. Terra Decisions - Interactive Mission Simulator
- **Immersive Decision-Making Gameplay**: Take on the role of Terra mission scientist
- **Multiple Mission Scenarios**:
  - Amazon Forest Watch (6-month observation strategy)
  - Pacific Ocean Anomaly (El Ni√±o monitoring)
  - Arctic Ice Crisis (polar region surveillance)
  - Sahara Dust Tracking (atmospheric aerosol monitoring)
  - Coral Reef Health (marine ecosystem observation)
- **5-Step Decision Trees**: Strategic choices with cascading consequences
- **Cumulative Scoring System**: Decisions build throughout missions with hidden consequences
- **Immediate Feedback**: Timeline-based consequence revelation
- **Detailed Results Screen**:
  - Mission rating (Excellent/Good/Moderate/Needs Work)
  - Complete decision timeline with explanations
  - Scientific insights and learning outcomes
  - Replay option for strategy exploration
  - Impact visualizations

### 3. Educational Assessment Quiz
- **Comprehensive Quiz Module**: Multi-section knowledge assessment
- **Question Categories**:
  - Terra Mission Basics (hardware, launch, orbit)
  - Instrument Overview (MODIS, ASTER, CERES, MISR, MOPITT)
  - Climate & Earth Science (observations, data usage)
  - Real-World Applications (disaster management, agriculture)
  - Advanced Topics (technical specifications, challenges)
- **Diverse Question Types**: Multiple choice, true/false, matching
- **Immediate Scoring**: Real-time feedback after completion
- **Performance Analytics**: Score breakdown by category
- **Learning Path**: Recommended topics for improvement
- **Progress Tracking**: Optional integration with learning management systems

## Installation & Setup

### Step 1: Clone Repository
```bash
git clone https://github.com/VicoKev/earth-insights.git
cd earth-insights
```

### Step 2: Install Dependencies
```bash
pnpm install
```

### Step 3: Verify Installation
```bash
pnpm run typecheck
pnpm run lint
```

### Step 5: Start Development
```bash
pnpm run dev
```

Application will be available at `http://localhost:5173`
